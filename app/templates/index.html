How to connect

  1. 1

    Create an org

    Visit /setup, enter your org name, and copy the API key shown.

  2. 2

    Set your API key

    Add LUMEN_API_KEY=your_key to your shell environment (e.g. ~/.zshrc).

  3. 3

    Add the hook to Claude Code

    Add this to your .claude/settings.json:

    {
  "hooks": {
    "PostToolUse": [{
      "matcher": "Skill",
      "command": "skill=$(cat | python3 -c \"import sys,json; d=json.load(sys.stdin); print(d.get('tool_input',{}).get('skill','unknown'))\" 2>/dev/null || echo unknown); curl --max-time 2 -s -X POST {{ base_url }}/events -H 'Content-Type: application/json' -H 'Authorization: Bearer $LUMEN_API_KEY' -d \"{\\\"event\\\": \\\"skill_used\\\", \\\"properties\\\": {\\\"skill\\\": \\\"$skill\\\"}, \\\"user\\\": \\\"$(git config user.email 2>/dev/null)\\\", \\\"project\\\": \\\"$(basename $(git rev-parse --show-toplevel 2>/dev/null) 2>/dev/null)\\\"}\" > /dev/null 2>&1 &"
    }]
  }
}
  4. 4

    Open your dashboard

    Visit /login, paste your API key, and events will start appearing as you use Claude Code.
